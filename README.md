## 🚀 How to Run Locally

1. Clone the repository
```bash
git clone https://github.com/Nirbhayjr/Nirbhay-Portfolio-main.git
cd Nirbhay-Portfolio-main
```

2. Start a local server (Python)
```bash
python -m http.server 8000
```

3. Open your browser and visit
```
http://localhost:8000
```

## 💡 Customization

### Update Skills
Edit `skills.json` to add/remove skills with their icons.

### Update Projects
Edit `projects/projects.json` to showcase your projects.

### Update Contact Information
Update email and social links in `index.html`.

### Modify Animations
Adjust timing in `assets/js/script.js` and `assets/css/style.css`.
